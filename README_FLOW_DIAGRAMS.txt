================================================================================
                          📋 PLATFORM OVERVIEW
================================================================================

CORE CONCEPT:
─────────────
Mavericks Ascend is a comprehensive training and talent management platform
that tracks fresh graduates (Mavericks) from onboarding through training to
final deployment to projects.

5 USER ROLES:
─────────────
1. Maverick (Trainee) - Fresh graduate being trained
2. HR - Manages profiles, pipelines, batches, and deployments
3. Super Admin - Full platform control + AI analytics
4. Trainer - Conducts training and enters assessment marks
5. Manager - Searches talent and requests deployments

KEY WORKFLOWS:
──────────────
1. Registration → Profile Review → Approval
2. Batch Creation → Assign Pipeline → Training Begins
3. Training Sessions → Assessments → Progress Tracking
4. Batch Completion → Status: Available
5. Talent Search → Deployment Request → Approval → Deploy
6. Project Complete → Release → On Bench

AI INTEGRATION:
───────────────
✨ Resume Parsing & Skill Extraction
✨ Pipeline Generation Suggestions
✨ Batch Grouping Recommendations
✨ Skill Proficiency Tracking
✨ Candidate-Role Matching
✨ Performance Insights & Predictions
💰 Cost Tracking & Budget Controls (Super Admin)


================================================================================
                        🔑 KEY FEATURES BY ROLE
================================================================================

MAVERICK:
• Self-registration with AI-powered resume parsing
• View training schedule and assessments
• Track skill proficiency levels
• View deployment status

HR:
• Review and approve profiles
• Create pipelines (training paths)
• Create batches and assign mavericks
• Schedule training sessions
• Manage deployments
• Platform analytics

SUPER ADMIN:
• All HR features
• User management (all roles)
• AI cost analytics and controls
• Audit logs and security
• Platform configuration

TRAINER:
• Conduct training sessions
• Create assessments
• Enter marks (Single/Bulk/Excel)
• View batch performance
• Track own statistics

MANAGER:
• Search for talent (AI-powered)
• Create deployment requests
• View assigned team
• Release from projects
• Deployment analytics
